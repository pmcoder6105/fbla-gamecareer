This package includes 35 low-poly 3D models, created in Blender and fully optimized for use in Unity.

9 Vehicles – each featuring detailed interiors and an animation system for lifting shuttles.

15 Firefighting Equipment Props – essential assets for fire rescue and emergency-themed environments.

3 Animated Characters – complete with walk, run, sprint, and idle animations.

4 Types of Vegetation – each available in both normal and burned versions, with three Levels of Detail (LOD) per type for optimal performance.
